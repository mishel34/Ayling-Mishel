Aprende los temas mas importantes

Estos son los temas mas importantes que debes conocer en CSS

CSS

es un lenguaje de diseño grafico para definir y crear la presentacion de un documento estructurado escrito en un lenguaje de marcado

html

es el componente mas basico de la Web. Define el significado y la estructura del contenido web.

base de datos

se encarga no solo de almacenar datos, sino tambien de conectarlos entre si en una unidad logica

programacion

es el proceso de crear un conjunto de instrucciones que le dicen a una computadora como realizar algun tipo de tarea. Pero no solo la accion de escribir un codigo para que la computadora o el software lo ejecute
